
## Sobre o projeto
Esté é um desafio para praticar Programação Orientada a Objetos do curso da EBAC - Front-End.
Os arquivos deste repositório são para a entrega do exercício do Módulo 7

## Funcionalidades exigidas:
- Receber um valor inserido pelo usário para calcular o tempo no estacionamento;
- O tempo será definido através do valor recebido.
  -- R$1,00 = 30 minutos;
  -- R$1,75 = 60 minutos;
  -- R$3,00 = 120 minutos (máximo permitido).
- Calcular e exibir o tempo correspondente e o troco (se houver);
- Exibir uma mensagem de "Valor insuficiente" caso o valor seja menor que R$1,00.

### Funcionalidades que foram colocadas de forma adicional:
- Cronômetro regressivo;
- Interface dinâmica com respostas em tempo real para o usuário;
- Mensagens de erro e sugestões de upgrade;
- Tratamento de dados para melhor experiência como:
  -- Valores nulos;
  -- Possíveis bugs;
  -- Textos dinâmicos e limpeza de capos;
  -- Classes separadas para executar funcionalidades específicas;
  -- Funções matemáticas dedicadas para calcular o troco de forma eficiente.
